Bienvenido al sistema de inventario DM

Las caracteristicas principales de este sistemas segun las especificaciones brindadas por el equipo de FFFF.

La API permitira:

  • Agregar / eliminar artículos de almacén y establecer su cantidad de stock. Los artículos tienen el siguiente información:
    • SKU: Identificador del artículo.
    • Descripción.
    • Cantidad en stock.
  • Cree un pedido seleccionando SKU y cantidad.
  • Cancelar un pedido.
  • Crear un pedido debería reducir la cantidad de SKU seleccionada, cancelar un pedido debería restaurar Cantidad de SKU.
  • Mantenga un registro de todos los pedidos (fecha ingresada, sku, cantidad, estado, quién creó el pedido).
  • Obtenga una lista de todos los SKU disponibles. (cantidad> 0), la lista debe tener paginación.
  • Buscar en artículo específico SKU.

Requerimientos de seguridad:

  • Agregar funcionalidad de inicio de sesión para:
    • Solo los administradores pueden agregar / eliminar / modificar SKU en el inventario.
  • Todos pueden crear pedidos y cancelar pedidos.
  • Todos pueden obtener una lista de todos los SKU.
  • Todos pueden usar la función de búsqueda.

Requerimientos de interfaz:

  • Cree una pequeña aplicación front-end que se conecta a la API para mostrar una lista de SKU.
  • La aplicación debe permitir buscar elementos por SKU.
  • Debe ser 100% receptivo.
  • Puedes hacerlo tan visualmente atractivo como puedas.
